Footwork is a frontend framework based on KnockoutJS that aims to be fully featured, expressive, and easy to use while remaining as idiomatic and compatible with Knockout and its code base as possible.
  • Custom elements/components and declarative view models
  • Automatic resolution and binding of declarative resources
  • Page routing with unlimited named outlet support
  • Transition effect and animation system including sequenced animations
  • Does not require a build system or module loader (but it works great with them)
  • Inter-module communication and state sharing
  • Much, much more...
Made With
KnockoutJS
Lodash

Why Footwork?

The aim is to enable the creation of fully featured single-page Knockout-based applications that are expressive, capable, and maintainable. Footwork provides the missing SPA-holes that knockout-only applications typically have, as well as providing some additional novel features not found in other frameworks.

Aimed at users or fans of KnockoutJS, if you are in that category you should definitely take a look.
